# Spinners Sass

## Getting Started
`npm install --save spinners-sass`

## Usage
HTML
```html
<div class="spinners-sass -skeleton"></div>
<div class="spinners-sass -round"></div>
```

SCSS
```bash
$primary-color: …
$box-size: …
@import "../../node_modules/spinners-sass/sass/spinners";
```
